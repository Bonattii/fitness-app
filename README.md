## Installation

#### You can use the application locally just by running npm run dev.

Clone the project

```bash
  git clone https://github.com/Bonattii/fitness-app
```

Go to the directory

```bash
  cd fitness-app
```

Install dependencies

```bash
  npm install
```

Start 

```bash
  npm run dev
```
